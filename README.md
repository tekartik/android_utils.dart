# android_utils.dart

Android utilities

## Setup

```
dependency:
  tekartik_android_utils:
    git:
      url: https://github.com/tekartik/android_utils.dart
    version: '>=0.1.0'
``` 
## Activation

### From git repository

```
pub global activate -s git https://github.com/tekartik/android_utils.dart
```

### From local path

```
dart pub global activate -s path .
```
