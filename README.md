# android_utils.dart

Android utilities

## Setup

```
dependency:
  tekartik_android_utils:
    git:
      url: git://github.com/tekartik/android_utils.dart
      ref: null_safety
    version: '>=0.1.0'
``` 
## Activation

### From git repository

```
pub global activate -s git git://github.com/tekartik/android_utils.dart
```

### From local path

```
pub global activate -s path .
```
